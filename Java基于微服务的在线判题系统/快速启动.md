# 拉取代码到本地仓库

# docker 镜像加速（可选）
# docker拉取镜像
```powershell
docker run -d --name oj-mysql -p 3307:3306 -e "TZ=Asia/Shanghai" -e "MYSQL_ROOT_PASSWORD=123456" mysql:5.7


```


# 然后修改各个服务的bootstrap.yml文件
```yml
spring:  
  application:  
    name: oj-system  
  profiles:  
    active: local  
  cloud:  
    nacos:  
      discovery:  
        namespace: nacos创建的bitoj命名空间链接  
        server-addr: http://localhost:8848  
#        username: nacos  
#        password: nacos  
        ip: 127.0.0.1  
      config:  
        namespace: nacos创建的bitoj命名空间链接  
        server-addr: http://localhost:8848  
        file-extension: yaml  
#        username: nacos  
#        password: nacos
```
