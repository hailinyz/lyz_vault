**使用无符号右移 >>>**  相当于除2
```java
/*  
 * 二分查找  
 */public static int binarySearch(int[] a, int target){  
    //设置左右边界  
    int i = 0,j = a.length - 1;  
  
    //循环查找  
    while (i <= j){  
        int mid = (i + j) >>> 1; //中间元素  
        if (target < a[mid]){  
            j = mid - 1;  
        } else if (a[mid] < target){  
            i = mid + 1;  
        }else {  
            return mid;  
        }  
          
    }  
    return -1;  
}
```
