## 1. 技术栈概述

### 1.1 前端技术栈

- **编程语言**：HTML/CSS/JavaScript
- **前端框架**：原生JavaScript + DOM操作
- **通信技术**：WebSocket（实时通信）、XMLHttpRequest/Fetch API（HTTP请求）
- **UI组件**：原生CSS样式 + 自定义组件

### 1.2 后端技术栈

- **编程语言**：Java
- **JDK版本**：JDK 17
- **后端框架**：Spring Boot 3.x
- **数据库**：MySQL 8.0+
- **ORM框架**：MyBatis-Plus
- **缓存系统**：Redis
- **消息队列**：RabbitMQ
- **通信协议**：HTTP RESTful API、WebSocket
- **构建工具**：Maven
- **认证框架**：JWT (JSON Web Token)
- **日志框架**：SLF4J + Logback

## 2. 环境配置详情

### 2.1 开发环境配置

- **IDE**：IntelliJ IDEA
- **构建工具**：Maven 3.8+
- **API测试工具**：Apifox
- **数据库管理工具**：Navicat

### 2.2 服务器环境配置

- **操作系统**：Linux (Ubuntu 22.04) / Windows Server
- **Java运行环境**：JRE 17
- **数据库服务**：MySQL 8.0+
- **缓存服务**：Redis 6.0+
- **消息队列服务**：RabbitMQ 3.8+
- **Web服务器**：内置Tomcat (Spring Boot)

## 3. 部署架构

### 3.1 开发环境

- **应用层**：Spring Boot应用 (端口: 8080)
- **数据层**：MySQL数据库 (端口: 3306)
- **缓存层**：Redis服务 (端口: 6379)
- **消息队列**：RabbitMQ服务 (端口: 5672, 管理界面: 15672)
- **前端静态资源**：通过Spring Boot内置服务器提供
